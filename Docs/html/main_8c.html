static void runApp ( void  )
static

Apos o termino da inicializacao do software feita pela funcao startApp(void) a task principal ficara neste loop infinito enquanto as Threads executam todo o processos. Para finalizar o SW e necessario alterar o status da aplicacao para APP_REQUEST_CLOSE atraves da funcao setAppStatus(TStatusApp sts_app) o loop sera interronpido e se iniciara o shutdown do SW atraves da funcao closeApp(void)
